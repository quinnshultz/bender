# Bender the Robot
A simple bending unit.

## Description
Got too much scrap laying around? Maybe this program can help.

Set the OPENAI_API_KEY environment variable and run the program:
```
export OPENAI_API_KEY=<yourkeyhere>
python head.py
```

## Prerequisites:
### Python Libraries: 
* bs4
* openai 0.28
* pocketsphinx 5.0.1
### OS libraries: 
* sox (Linux)
